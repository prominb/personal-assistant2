Програма працює у консольному режимі та пропонує користувачу рядок опцій для взаємодії з нотатками. Вона використовує файл notes.json для
 зберігання інформації про нотатки. Ось як може працювати код:

Користувач запускає програму.
Програма виводить меню опцій:
markdown
Copy code
1. Додати нотатку
2. Пошук нотаток
3. Редагувати нотатку
4. Видалити нотатку
5. Вийти
Користувач вибирає опцію, введену відповідний номер.
Відповідно до обраної опції виконується відповідна функція.
Якщо користувач вибирає "Додати нотатку", програма запитує заголовок, текст та теги нової нотатки і додає їх у список нотаток.
Якщо користувач вибирає "Пошук нотаток", програма запитує запит для пошуку та виводить знайдені нотатки.
Якщо користувач вибирає "Редагувати нотатку", програма запитує заголовок нотатки, яку користувач хоче відредагувати, та надає можливість 
введення нового заголовка, тексту та тегів.
Якщо користувач вибирає "Видалити нотатку", програма запитує заголовок нотатки, яку користувач хоче видалити, і видаляє відповідну нотатку 
зі списку.
Якщо користувач вибирає "Вийти", програма завершує виконання.
Програма зберігає зміни у файлі notes.json під час виконання операцій додавання, редагування та видалення нотаток.
